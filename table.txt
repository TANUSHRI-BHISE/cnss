OSI	Open Systems Interconnection	7-layer network model
LAN	Local Area Network	Small local network (office/home)
WAN	Wide Area Network	Large-scale network (Internet)
NIC	Network Interface Card	Connects computer to network (Layer 2)
Hub	—	Physical device that broadcasts all data
Switch	—	Forwards frames using MAC addresses (Layer 2)
Router	—	Routes packets using IP (Layer 3)
IP	Internet Protocol	Logical addressing & packet routing
IPv4	Internet Protocol version 4	32-bit addressing (e.g., 192.168.1.1)
IPv6	Internet Protocol version 6	128-bit addressing (e.g., 2001:db8::1)
NAT	Network Address Translation	Converts private IP ↔ public IP
CIDR	Classless Inter-Domain Routing	Efficient IP allocation using prefix (e.g., /24)
ARP	Address Resolution Protocol	Maps IP → MAC address
RARP	Reverse Address Resolution Protocol	Maps MAC → IP address
ICMP	Internet Control Message Protocol	Error & control messages (e.g., ping)
IGMP	Internet Group Management Protocol	Manages multicast group membership
RIP	Routing Information Protocol	Distance Vector Routing (uses hop count)
OSPF	Open Shortest Path First	Link State Routing (uses Dijkstra’s algo)
BGP	Border Gateway Protocol	Path Vector Routing (used between ISPs)
MPLS	Multiprotocol Label Switching	Label-based fast packet forwarding
AODV	Ad hoc On-Demand Distance Vector	MANET routing, on-demand path creation
DSR	Dynamic Source Routing	MANET routing, source keeps full path
MANET	Mobile Ad-hoc Network	Wireless, infrastructure-less network
AS	Autonomous System	Independent network under one admin (used in BGP)
QoS	Quality of Service	Controls latency and priority in networks
MTU	Maximum Transmission Unit	Largest packet size that can be sent
TTL	Time To Live	Limits packet lifetime in network
MAC	Media Access Control	Hardware address at Data Link layer
ISP	Internet Service Provider	Provides Internet connectivity
FQDN	Fully Qualified Domain Name	Complete domain name (e.g., www.example.com
)

✅ Viva Quick-Tips:

RIP = hop count, OSPF = shortest path, BGP = policy routing

ARP ↔ MAC, RARP ↔ IP

NAT = private ↔ public IP, CIDR = flexible subnetting

AODV / DSR = mobile networks (MANET)
