ü™õ Basic Setup Questions
Q	A
What is a LAN?	LAN (Local Area Network) connects computers within a small area like a lab or office.
What is a Wired LAN?	A LAN where computers are connected using physical cables (Ethernet).
What is a Switch?	A Layer 2 device that connects multiple devices in a network and forwards data based on MAC addresses.
What is the difference between a Hub and a Switch?	A hub sends data to all ports (broadcasts), while a switch sends data only to the destination port (intelligent forwarding).
Why do we use a Switch instead of a Hub?	Because switches are faster and reduce unnecessary traffic by using MAC address tables.
üíª IP Addressing Questions
Q	A
What is an IP address?	A unique logical address assigned to every device on a network.
What is a Subnet Mask?	It defines which part of the IP address is the network portion and which is the host portion.
What is the IP version used here?	IPv4 (e.g., 192.168.1.10).
Can two devices have the same IP address?	No, it causes an IP conflict and communication failure.
What does 255.255.255.0 mean?	It‚Äôs a subnet mask indicating a Class C network, allowing up to 254 hosts.
üì° Ping & Connectivity Questions
Q	A
What is the purpose of the ping command?	To test connectivity between two devices.
Which protocol does ping use?	ICMP (Internet Control Message Protocol).
What are the two ICMP message types used in ping?	Echo Request and Echo Reply.
What happens if ping fails?	It indicates a network or configuration problem (wrong IP, cable issue, etc.).
What layer of OSI model does ping work on?	Network Layer (Layer 3).
üîç Wireshark / Simulation Mode Questions
Q	A

What does Simulation Mode do in Packet Tracer?	It lets you view and analyze how data packets flow step-by-step.
What details can you see in a packet capture?	Source and Destination MAC, IP addresses, ICMP type (Request/Reply), TTL, etc.
What layers of the OSI model can you see in Simulation Mode?	Data Link (Ethernet), Network (IP), and Transport/ICMP layers.

üìö Conceptual / OSI Model Questions
